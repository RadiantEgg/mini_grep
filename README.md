# mini_grep

## 这是基于C语言实现的 Unix 系统命令行工具 grep 的简化版本

### 命令行工具用法
- ./build/mini_grep [可选项] [目标内容] [文件1] [文件2] ...

### 支持简单文件内容查找，可选项有
- ```-n```: 输出包含目标内容的行所在文件行数
- ```-x```: 反向输出，即输出不包含目标内容的行
- ```-i```: 忽略目标内容和文件内容大小写进行查找
- 可组合使用 :```-nxi / -xn / -x -i / -n -x```

### 支持多文件查找
